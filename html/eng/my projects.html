www.nathaliaklatovski.cz

Website where you will find lots of beautiful original poetry. The site is written in Python (Flask), HTML, CSS and JS and uses a database PostgreSQL. The page has its own admin, which the author can use herself to add, remove and edit content anytime and from anywhere. The site is deployed on Heroku.

Ikea Scrapper

The application checks IKEA products on a daily basis based on URLs and prices entered in a simple GoogleSheet. If the product falls below the required threshold, it sends the user an e-mail with information about the price change. The modules used are BeautifulSoup, Selenium, the application communicates with GoogleSheet via the Sheety API.

Stack

Python

Mailing Tool

It is an Excel application written in VBA. The application connects to the user's Outlook application. It automates the sending of emails to different addresses based on the key. Can send data in email body, attachments, and more. Mailing can be easily prepared from the user's XLS file. Email replies can be retrieved. XLS Tables can then merge into one "MASTER" table.
